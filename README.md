# 贝特猫IM (BetterMao IM)

贝特猫IM是一个仿微信的即时通讯系统，基于Node.js和Vue.js开发，支持实时消息、好友添加、群聊等功能。

## 功能特性

- **用户系统**：注册、登录、个人信息管理
- **实时消息**：基于WebSocket的实时消息传递
- **好友系统**：添加好友、好友列表
- **群聊功能**：创建群聊、群成员管理
- **消息管理**：发送消息、撤回消息、标记已读
- **响应式设计**：适配不同屏幕尺寸
- **现代化界面**：采用Apple风格的简洁设计

## 技术栈

### 后端
- Node.js + Express.js：提供API接口
- Socket.io：实现WebSocket实时通信
- SQLite：轻量级数据库存储

### 前端
- Vue.js 2.x：前端框架
- 原生CSS：样式设计
- 响应式布局：适配不同设备

## 项目结构

```
├── im-platform/     # 后端API服务
```

## 快速开始

### 环境要求
- Node.js 14.0+
- npm 6.0+

### 安装依赖

```bash
npm install
```

### 启动服务

1. 启动后端API服务

```bash
cd im-platform
node index.js
```

2. 启动WebSocket服务

```bash
cd im-server
node index.js
```

3. 启动前端本地服务器

```bash
cd im-web
python -m http.server 8000
```

### 访问系统

打开浏览器，访问：`http://localhost:8000`

## 数据库

项目使用SQLite数据库，数据库文件位于 `im-platform/im_platform.db`。

首次启动时会自动创建数据库表结构和测试数据。
